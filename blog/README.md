# vue-blog

 用户界面部分

### 功能

- 导航 √
- 阅读量 √
- 归档时间轴 √
- 文章类型转换 √
- 标签分类 √
- 评论 √
- 评论表情
- 文章列表下拉加载 √

### 命令

- 启动调试
 ```
 npm run dev
 ```
- 打包
 ```
 npm run build
 ```
- 打包并生成 report.html 文件（用于性能分析）
 ```
 npm run build:report
 ```
- 优化 svg
 ```
 npm run svgo
 ```
